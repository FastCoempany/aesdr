## Production Notes

### Tone
- Direct. Conversational. Like talking to a friend at a bar, not presenting at a conference.
- No hype words: no "amazing," "incredible," "life-changing," "game-changer."
- Slight weariness is okay — this person has been in the trenches and isn't performing enthusiasm.

### Visual Options (low-budget)

**Option A: Screen recording + voiceover**
- Record scrolling through the AESDR course UI (lesson cards, interactive exercises)
- Voiceover reads the script
- End card: aesdr.com + price + refund guarantee
- Cost: $0 (just need a mic)

**Option B: Talking head**
- Founder on camera, clean background, natural lighting
- Direct-to-camera delivery. No teleprompter feel — memorize the key beats, ad-lib the rest
- End card: same as above
- Cost: $0-50 (phone camera + ring light)

**Option C: Text-on-screen with music**
- Key phrases appear on dark background (AESDR brand colors)
- Lo-fi or ambient background music
- End card: same
- Cost: $0 (CapCut or Canva)

### Technical
- **Resolution:** 1920x1080 minimum
- **Format:** MP4, H.264
- **Max file size:** Google Ads accepts up to 256MB
- **Aspect ratios:** 16:9 (standard), 9:16 (Shorts/Stories), 1:1 (optional)
